# todos-react
TODO list page implementation in [TypeScript](https://github.com/Microsoft/TypeScript) using [React](https://github.com/facebook/react)

## Build
### Install dependencies
```bash
$ yarn install
```

### Compile
```bash
$ npm run build
```

## Test
```bash
$ npm run test
```

## Run
```bash
$ node run start
```
